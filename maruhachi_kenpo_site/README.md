# 丸八真綿健康保険組合 ホームページ

## サイト構成

このサイトは丸八真綿健康保険組合の公式ホームページです。

### ディレクトリ構造

```
maruhachi_kenpo_site/
├── index.html                    # トップページ
├── about/
│   └── index.html               # 健保組合のご案内
├── benefits/
│   └── index.html               # 病気やケガをしたとき（給付金について）
├── health/
│   └── index.html               # 疾病予防・健康づくり（保健事業）
├── forms/
│   └── index.html               # 届出申請書一覧
└── contact/
    └── index.html               # お問い合わせ

```

## 主要ページ一覧

1. **トップページ** (`index.html`)
   - お知らせ表示
   - 主要メニューへのアクセス（アイコンカード）
   - 重要情報の表示

2. **健保組合のご案内** (`about/index.html`)
   - 組合の概要
   - 組織情報
   - 財政状況
   - 運営体制

3. **病気やケガをしたとき** (`benefits/index.html`)
   - 療養の給付
   - 高額療養費
   - 傷病手当金
   - 出産育児一時金
   - 出産手当金
   - 埋葬料
   - その他各種給付

4. **疾病予防・健康づくり** (`health/index.html`)
   - 定期健康診断
   - 人間ドック（補助30,000円）
   - レディース健診（新設・補助10,000円）
   - 被扶養者健診（補助10,000円）
   - インフルエンザ予防接種（補助3,000円）
   - 特定健診・特定保健指導

5. **届出申請書一覧** (`forms/index.html`)
   - 各種申請書のダウンロード
   - カテゴリー別フィルター機能
   - 19種類の申請書を掲載

6. **お問い合わせ** (`contact/index.html`)
   - 連絡先情報
   - お問い合わせフォーム
   - アクセス情報

## 実装されている機能

### 共通機能
- レスポンシブデザイン（スマートフォン対応）
- 文字サイズ変更機能（小・中・大）
- 統一されたヘッダー・フッター
- パンくずリスト
- 検索ボックス（デザインのみ）

### ページ固有機能
- **申請書一覧**: カテゴリー別フィルター
- **お問い合わせ**: フォーム送信機能（デモ版）
- **保健事業**: 令和7年度の変更点を明示

## 令和7年度の主な変更点

以下の保健事業が拡充されました：

1. **被扶養者健診補助**: 7,000円 → **10,000円**
2. **レディース健診**: **新設（10,000円）**
3. **インフルエンザ予防接種補助**: 2,000円 → **3,000円**
4. **人間ドック補助**: 25,000円 → **30,000円**

## デザインの特徴

- 青色（#0066cc）をメインカラーとした清潔感のあるデザイン
- アイコンを活用した視覚的な分かりやすさ
- カード型レイアウトで情報を整理
- ホバーエフェクトによる操作性の向上

## 使用方法

1. ブラウザで `index.html` を開く
2. 各ページへのリンクをクリックして移動
3. レスポンシブデザインのため、スマートフォンでも表示可能

## 注意事項

- このサイトは静的HTMLサイトです
- フォーム送信機能は現在デモ版です（実際の送信機能は未実装）
- 申請書PDFのダウンロードリンクは現在プレースホルダーです
- 検索機能は現在デザインのみです
